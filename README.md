# E-commerce Laravel Project

## 👩‍💻 Author
Naira Mohamed

## 📌 Description
This is a full-stack e-commerce web application built using Laravel.
The project includes both front-end and back-end functionalities.

## 🚀 Features
- User authentication system
- Product listing and categories
- Shopping cart
- Admin dashboard (Voyager)
- Order management system

## 🛠️ Technologies Used
- Laravel
- PHP
- MySQL
- HTML / CSS / JavaScript
- Bootstrap

## ⚙️ Installation
1. Clone the repo
2. Run: composer install
3. Copy .env file
4. Run: php artisan key:generate
5. Setup database
6. Run: php artisan serve
